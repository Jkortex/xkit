# Daily TUI 交互指南

`daily tui` 是一个高效的终端交互界面，用于管理、搜索和朗读备忘录。

---

## 1. 页面与标签页

TUI 界面由四个选项卡组成：

| 按键 | 标签页                | 说明                                                       |
| :--- | :-------------------- | :--------------------------------------------------------- |
| `1`  | **Inbox (活跃)**      | 当前所有未归档备忘录，支持全部操作                         |
| `2`  | **Archived (归档)**   | 已归档备忘录，可查看或恢复                                 |
| `3`  | **Tags list (标签)**  | 列出所有标签及其备忘录计数，`enter` 可按标签过滤跳转 Inbox |
| `4`  | **Statistics (统计)** | 展示备忘录 / 标签 / 资源的计数报表                         |

**标签页切换**：

- `h` / `l` 或 `←` / `→`：在相邻标签页间循环
- `1` ~ `4`：直接跳转

---

## 2. 列表导航

| 按键      | 功能                                   |
| :-------- | :------------------------------------- |
| `j` / `↓` | 光标下移                               |
| `k` / `↑` | 光标上移                               |
| `g`       | 跳转到列表顶部                         |
| `G`       | 跳转到列表底部                         |
| `空格`    | 展开 / 折叠当前备忘录全文              |
| `enter`   | 进入**细节阅读模式**（单窗格全屏浏览） |

---

## 3. 编辑与创建

| 按键 | 功能                                        |
| :--- | :------------------------------------------ |
| `c`  | 在底部输入行快速创建纯文本备忘录            |
| `C`  | 唤醒外部编辑器创建新备忘录（支持 Markdown） |
| `e`  | 唤醒外部编辑器编辑当前选中备忘录            |
| `T`  | 编辑标签（空格分隔，支持批量选中后修改）    |

> 编辑器优先级：`$EDITOR` 环境变量 → `nvim` → `vim` → `code` → `notepad.exe` (Windows) / `nano` → `vi` (Linux/macOS)。
> 编辑器路径在 TUI 启动时后台预解析，首次打开无延迟。

---

## 4. 删除与归档

| 按键 | 功能                                                 |
| :--- | :--------------------------------------------------- |
| `d`  | 删除当前 / 批量选中的备忘录（底部显示 `y/n` 确认）   |
| `a`  | 归档（Inbox 中）或恢复（Archived 中）                |
| `x`  | 勾选 / 反选当前备忘录，配合 `d` `a` `T` 进行批量操作 |

---

## 5. 音频朗读 (TTS)

| 按键 | 功能                                                   |
| :--- | :----------------------------------------------------- |
| `p`  | 朗读当前选中备忘录的完整正文（自动清除 Markdown 格式） |
| `s`  | 显示当前备忘录的 TTS 缓存状态                          |
| `S`  | 打开 TTS 配置面板                                      |

**细节阅读模式下**按 `p` 可朗读当前高亮的 Markdown 段落块。

---

## 6. 搜索与筛选

### 6.1 快速过滤

| 按键  | 功能                                                                                 |
| :---- | :----------------------------------------------------------------------------------- |
| `t`   | 在已有标签中循环切换过滤（最后一次清除）                                             |
| `r`   | **重置**：清除**所有**过滤器（搜索词、标签、日期、排序、选中状态），重新加载默认列表 |
| `esc` | 清除当前快速过滤状态                                                                 |
| `y`   | 复制当前备忘录内容到剪贴板                                                           |
| `?`   | 在底部显示快捷键提示                                                                 |

### 6.2 DSL 过滤模式 (按 `:` 进入)

在列表页按 `:` 激活底部命令行输入。输入表达式后按 `enter` 执行过滤。

| 命令格式          | 说明                            | 示例                   |
| :---------------- | :------------------------------ | :--------------------- |
| `/文本`           | 搜索正文包含该文本的备忘录      | `/golang`              |
| `t:标签1,标签2`   | 匹配**任意一个**标签（OR）      | `t:work,lingo`         |
| `t+:标签1,标签2`  | 必须**同时拥有**所有标签（AND） | `t+:work,important`    |
| `t-:标签`         | **排除**拥有该标签的备忘录      | `t-:personal`          |
| `from:日期`       | 创建时间 ≥ 该日期               | `from:2026-06-01`      |
| `to:日期`         | 创建时间 ≤ 该日期               | `to:2026-06-05`        |
| `sort:规则`       | 排序方式                        | `sort:updated_at_desc` |
| `l:数量`          | 限制显示条数                    | `l:10`                 |
| `clear` / `reset` | 清除全部筛选器                  | `clear`                |

**标签自动补全**：输入 `t:` 等标签命令时，底部会根据当前输入实时提示匹配的标签。支持多值场景（如 `t:tag1,ta`，会对最后一个逗号后的子串进行补全提示）。

**排序规则可选值**：`created_at` / `created_at_asc` / `created_at_desc` / `updated_at` / `updated_at_asc` / `updated_at_desc`

---

## 7. 细节阅读模式 (Detail Mode)

在列表中按 `enter` 进入。内容按 Markdown 语义拆分为逻辑块（标题、段落、列表、代码块），当前块高亮显示。

| 按键          | 功能                       |
| :------------ | :------------------------- |
| `j` / `↓`     | 下一个段落块               |
| `k` / `↑`     | 上一个段落块               |
| `p`           | 朗读当前高亮段落块         |
| `y`           | 复制当前段落块文本到剪贴板 |
| `e`           | 编辑此备忘录               |
| `esc` / `tab` | 返回列表                   |

宽屏 (≥ 80 列) 下自动启用**左右分栏**：左侧列表、右侧预览。

---

## 8. TTS 配置面板 (按 `S` 进入)

| 操作       | 按键                          |
| :--------- | :---------------------------- |
| 导航       | `↑` / `↓`                     |
| 编辑文本项 | `enter` → 输入 → `enter` 保存 |
| 切换选项项 | `←` / `→` 或 `enter` 循环     |
| 切换布尔值 | `s`                           |
| 返回列表   | `esc`                         |

配置修改后自动持久化至 `~/.daily/config.json`。

---

## 9. 命令面板 (按 `:` 后可见)

命令面板整合了常用操作，支持模糊搜索和上下选择：

| 按键                             | 功能             |
| :------------------------------- | :--------------- |
| 输入文字                         | 模糊过滤命令列表 |
| `↑` / `↓` 或 `Ctrl+p` / `Ctrl+n` | 在结果中导航     |
| `enter`                          | 执行选中的命令   |
| `esc`                            | 退出命令面板     |

---

## 10. 退出

| 按键     | 功能     |
| :------- | :------- |
| `q`      | 退出 TUI |
| `Ctrl+C` | 强制退出 |
