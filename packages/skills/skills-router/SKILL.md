# Skills Router (技能路由器)

本技能在 Session 启动时以“系统预置”形式加载，作为智能体调度其他特定领域技能的引擎。

## 运作规程

### 1. 初始化会话 (Session Start)

- 启动时立即读取同级目录下的 `router.json`。
- 将所有可用的技能名、触发关键词及文件路径映射常驻内存。

### 2. 运行时动态路由匹配

- **输入扫描**：在分析用户输入的任务时，主动对齐触发关键词。
- **动态读取**：一旦发现用户的任务意图契约（例如 "写个接口设计" 命中了 "api-design" 的触发词 "接口设计"），立即调用工具读取 `./api-design/SKILL.md` 的内容。
- **上下文合并**：将匹配技能的具体开发指南，无缝合并至当前的执行逻辑中，强制执行规范。
